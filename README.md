# BFSI_Portfolio_Risk_Analytics_SQL_Project
📊 Credit Risk | NPA Analysis | Collection Efficiency | Fraud Signals

Author: Deepak Vishwakarma
Role: Finance Professional | Portfolio Analytics
Tools: SQL, Power BI

🔍 Project Overview

This project simulates a real-world BFSI loan portfolio analytics system, focusing on key financial and risk management areas such as:

NPA (Non-Performing Asset) tracking
Credit risk assessment (CIBIL, DPD, LTI)
Collection efficiency monitoring
Fraud signal detection
Branch-level performance analysis

The objective is to move beyond traditional Excel MIS and build a data-driven decision-making system using SQL and dashboards.
